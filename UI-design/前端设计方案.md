# 目标：视频会议管理系统 UI 重新设计

# 指令：
# 启用 `UI UX Pro Max` skill。

# 核心愿景：
将 `1.jpg` 到 `6.png` 所描述的视频会议管理系统的 UI 重新设计为一个专业、沉浸式、无缝的 Glassmorphism 2.0 风格界面。核心理念是提高清晰度、视觉层次感、专业感和沉浸感，并结合微妙的新极简主义细节。所有原始文本和数据点必须被保留并正确显示。

# 统一视觉指南：

## 颜色调色板 (深色 Indigo 主题)：
* **主色 (Indigo Blue):** 深 Indigo (#2E3A59)，用于所有主要强调色、按钮和侧边栏背景（替代原有的纯蓝和深灰）。
* **强调色:**
    * **Teal Green:** (#1E9E6F) 用于成功、待开始状态。
    * **Amber Yellow:** (#FBC02D) 用于进行中状态。
    * **Soft Crimson Red:** (#E57373) 用于已结束状态（谨慎使用）。
* **背景 (Light Cool Grey):** 用于整个页面背景 (#F4F6F8)。
* **文本:**
    * **Indigo Blue Text:** (#1A1F3B) 用于标题和描述（在浅色背景上）。
    * **White Text:** 用于深色背景（侧边栏）。

## 卡片设计：
* 所有卡片都将采用微妙的 glassmorphism，具有 subtle blur (#FFFFFF 20% alpha) 和 soft shadow (0 4px 12px rgba(0,0,0,0.05))。
* 使用 Neumorphic 软凹陷细节来指示输入框和选定元素。

## 版式：
* 全面使用 Inter 字体。增加字体粗细和大小的对比：
    * Heading 1: Large, bold (2.5rem)
    * Heading 2: Medium-large (2rem)
    * Heading 3: Medium (1.5rem)
    * Body: Regular (1rem)
    * Caption: Small (0.875rem)

## 图标：
* 使用 Heroicons 图标，现代、干净且一致。
